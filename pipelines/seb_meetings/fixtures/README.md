# SEB meetings fixtures

## `seb_meetings_v0.xlsx`

The hand-built workbook that pre-dates this repo. It is the source of truth for the schema until live ingestion replaces it, and a permanent reference for the human-review patterns this pipeline must support.

**Sheets:** Dashboard, Meetings, Controversies, Sources, Update Script
**Coverage:** Feb 2025 – May 2026 (17 meetings)
**Origin:** Author-maintained tracker, January 2026 onward

Do not edit this file as part of pipeline development. If the schema needs to change, change the model (`transforms/models.py`) and the warehouse schema (`warehouse/schema/seb.sql`), then update the workbook deliberately as its own PR.

## `seb_youtube_rss.xml`

A captured payload from the live SEB channel's Atom feed:

    https://www.youtube.com/feeds/videos.xml?channel_id=UC3t-f42tkjx9lXejVcWRzrg

Kept as a test fixture so `test_youtube_rss.py` runs offline and so changes to YouTube's feed shape are caught by failing tests rather than mysterious production breakage. Refresh occasionally with:

    curl -sL "https://www.youtube.com/feeds/videos.xml?channel_id=UC3t-f42tkjx9lXejVcWRzrg" \
      -o pipelines/seb_meetings/fixtures/seb_youtube_rss.xml

After refresh, expect a few test assertions about the most-recent video to need updating — that's the contract working as intended.
